1 Local environment setup

To use OriginHub over SSH, you need to generate an SSH key and add it to your account. Follow these steps for local development (default SSH port: 2222).

Step 1 — Generate SSH key

Create an ED25519 key pair. Skip this if you already have one. 

ssh-keygen -t ed25519 -C "your-email@example.com"

Step 2 — Copy your public key

Display your public key and copy it to the clipboard.

cat ~/.ssh/id_ed25519.pub

Step 3 — Add key to OriginHub

Paste the key in Settings → SSH Keys, or add via API:

Step 4 — Clone without SSH Config

git clone ssh://git@localhost:2222/owner/repo.git

3 SSH config (optional)

Alternative: you can add an entry to ~/.ssh/config for a shorter alias. 

Host originhub-local
    HostName localhost
    Port 2222
    User git
    IdentityFile ~/.ssh/id_ed25519

Then clone and push using the short host alias:

git clone originhub-local:username/repo-name.git
cd repo-name
git push origin main

2 Clone & push quick reference

Action Command
Clone (full URL) git clone ssh://git@localhost:2222/owner/repo.git
Clone (with config) git clone originhub-local:owner/repo.git
Push to branch git push origin main
Push new branch git push -u origin feature-branch
Pull latest git pull origin main
